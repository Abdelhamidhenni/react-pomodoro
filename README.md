# React Pomodoro

**1 -** Prepare the environment:

```sh
$ npm install -g webpack
```

**2 -** Clone the project and install the dependencies:

```sh
$ git clone https://github.com/afonsopacifer/react-pomodoro.git
$ cd react-pomodoro
$ npm install
```
**3 -** Run webpack and webpack-dev-server:

```sh
$ npm start
```
